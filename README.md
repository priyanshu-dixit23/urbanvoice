# 🏙️ UrbanVoice
> Your Voice. Your City. Your Change.

AI-powered civic complaint platform enabling citizens to report 
local issues via voice or text with instant AI-generated government responses.

## 🚀 Features
- 🎤 Voice recording with Groq Whisper AI
- 🤖 Instant AI government responses via LLaMA 3
- 📍 Auto geolocation detection
- 📊 Real-time issue dashboard
- 💾 Supabase database

## 🛠️ Tech Stack
- React, Node.js, Express
- Groq AI (Whisper + LLaMA 3)
- Supabase
- Browser Geolocation API
